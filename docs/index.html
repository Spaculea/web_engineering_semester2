src/db/initialization.js

Dieses Skript importiert alte Klausuren und Lösungen in eine PostgreSQL-Datenbank. Es stellt sicher, dass die Datenbank bereit ist, bevor die Importe gestartet werden. Es liest PDF-Dateien aus den angegebenen Verzeichnissen, extrahiert relevante Informationen aus den Dateinamen und fügt die Daten in die entsprechenden Tabellen ein.
